About

This project is a tool that uses light-dependent resistors

that can be used to measure from what direction the user is waving from.

Components

In this project I used several components:

- 9 Light-dependent resistors

- 9 1uF Capacitors

- 1 RaspberryPi 4 Model B

- 6 Half-sized breadboards

- Male-Female Pins

- Male-Male Pins
